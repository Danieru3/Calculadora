Calculadora Mobile com Expo & TypeScript

Este é um projeto de aprendizado desenvolvido passo a passo para criar uma calculadora funcional utilizando **React Native**, Expo e TypeScript.

Em Desenvolvimento (Entrega Parcial)

Até o momento, o projeto conta com:
- [x] Configuração do ambiente com Expo Router.
- [x] Criação da interface base (Display e Teclado).
- [x] Lógica de entrada de números (0-9) com tratamento de estado (`useState`).
- [x] Função para limpar a tela (Botão C).
- [x] Organização visual dos botões em grades (Rows).

Tecnologias Utilizadas

- TypeScript: Para tipagem estática e segurança do código.
- React Native / Expo: Framework para desenvolvimento cross-platform.
- GitHub Desktop: Para controle de versão e commits.

📸 Prévia Atual
Atualmente, a calculadora possui o layout numérico organizado e o visor funcional para entrada de dados simples.
